🎯 基本操作方法

スマートフォンでの視聴

FANZA Swipeはスマートフォンに最適化されたTikTok風インターフェースです。縦持ちでの視聴を推奨します。

動画の再生

動画は自動的に再生されます。音量は画面右下の音量ボタンで調整できます。

ヘッダー機能

画面上部には検索ボタン、作品詳細ボタン、メニューボタンがあります。

<\!-- スワイプ操作 -->

👆 スワイプ操作詳細

⬆️ 上スワイプ

次の動画に移動

画面を上方向にスワイプすると次の動画が表示されます。

⬇️ 下スワイプ

前の動画に戻る

画面を下方向にスワイプすると前の動画に戻ります。

👆 タップ操作

一時停止/再生

画面中央をタップすると動画の一時停止・再生ができます。

🔄 長押し

早送り再生

画面を長押しすると早送りで再生されます。

<\!-- 5動画システム -->

🎬 5動画システム

📊 5動画ごとの区切り

FANZA Swipeでは5つの動画を視聴すると「次の動画を見る」ボタンが表示されます。

解除方法

🔴 方法1: ボタンをタップ

「次の動画を見る」ボタンを直接タップして次の5動画を読み込みます。

⬇️ 方法2: 強いプル・トゥ・リフレッシュ

画面を強く下に引っ張ると、ボタンを押さずに自動的に次の動画群が読み込まれます。

<\!-- 検索機能 -->

🔍 検索機能の使い方

🔍 検索ボタン

画面右上の虫眼鏡アイコンをタップすると検索パネルが開きます。

検索方法

  • • キーワード検索: 作品タイトルや女優名で検索
  • • ジャンル検索: カテゴリーから選択
  • • 検索履歴: 過去の検索が記録されます
  • • 検索解除: ヘッダーの「キーワード ✕」をタップ
<\!-- 作品詳細 -->

📄 作品詳細・購入方法

ℹ️ 作品詳細ボタン

ヘッダー右上の「作品詳細」ボタンをタップすると詳細情報が表示されます。

詳細情報の内容

  • • 作品タイトル・説明
  • • 出演女優情報
  • • 再生時間・ジャンル
  • • レビュー評価・いいね数
  • • 販売価格情報
  • • セール終了日時
  • • FANZA商品ページリンク
  • • アフィリエイト購入ボタン

🛒 購入方法

「今すぐ購入」ボタンから直接FANZAの商品ページに移動して購入できます。

<\!-- プル・トゥ・リフレッシュ -->

⬇️ プル・トゥ・リフレッシュ

🔄 機能概要

画面を下方向に強く引っ張ることで、新しい動画を読み込むことができます。

使用場面

  • • 5動画制限時: ボタンを押さずに次の動画群を読み込み
  • • 動画更新時: 最新の動画を取得
  • • 接続エラー時: 再読み込みを実行

💡 コツ: 軽く下に引っ張るだけでは動画が戻るだけです。しっかりと強く引っ張ってください。

<\!-- 便利な使い方のコツ -->

💡 便利な使い方のコツ

⚡ 効率的な視聴方法

  • • 興味のない動画は素早く上スワイプでスキップ
  • • 気に入った動画は作品詳細で完全版を確認
  • • 検索機能でお気に入り女優の作品を探す
  • • プル・トゥ・リフレッシュで新着動画をチェック

🔧 トラブル対処法

  • • 動画が読み込まれない → プル・トゥ・リフレッシュ
  • • 音が出ない → 画面下の音量ボタンを確認
  • • 動作が重い → ブラウザのキャッシュを削除
  • • 検索結果が表示されない → キーワードを変更
