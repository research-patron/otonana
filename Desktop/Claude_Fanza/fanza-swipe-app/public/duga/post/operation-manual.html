🎯 基本操作方法

スマートフォンでの視聴

DUGA Swipeはスマートフォンに最適化されたTikTok風インターフェースです。縦持ちでの視聴を推奨します。

動画の再生

動画は自動的に再生されます。DUGAの高品質なアマチュア・個人撮影動画をお楽しみください。

ヘッダー機能

画面上部には検索ボタン、作品詳細ボタン、メニューボタンがあります。紫色のDUGAテーマになっています。

<\!-- スワイプ操作 -->

👆 スワイプ操作詳細

⬆️ 上スワイプ

次の動画に移動

画面を上方向にスワイプすると次のDUGA動画が表示されます。

⬇️ 下スワイプ

前の動画に戻る

画面を下方向にスワイプすると前の動画に戻ります。

👆 タップ操作

一時停止/再生

画面中央をタップすると動画の一時停止・再生ができます。

🔄 長押し

早送り再生

画面を長押しすると早送りで再生されます。

<\!-- 5動画システム -->

🎬 5動画システム

📊 5動画ごとの区切り

DUGA Swipeでは5つの動画を視聴すると「次の動画を見る」ボタンが表示されます。

解除方法

🟣 方法1: ボタンをタップ

「次の動画を見る」ボタンを直接タップして次の5動画を読み込みます。

⬇️ 方法2: 強いプル・トゥ・リフレッシュ

画面を強く下に引っ張ると、ボタンを押さずに自動的に次の動画群が読み込まれます。

<\!-- 検索機能 -->

🔍 検索機能の使い方

🔍 検索ボタン

画面右上の虫眼鏡アイコンをタップすると検索パネルが開きます。

DUGA検索の特徴

  • • キーワード検索: 作品タイトルやメーカー名で検索
  • • カテゴリー検索: M男、アイドル、フェチなどで絞り込み
  • • 個人撮影検索: アマチュア作品を重点的に検索
  • • 検索解除: ヘッダーの「キーワード ✕」をタップ
<\!-- 作品詳細 -->

📄 作品詳細・購入方法

ℹ️ 作品詳細ボタン

ヘッダー右上の「作品詳細」ボタンをタップすると詳細情報が表示されます。

DUGA詳細情報の内容

  • • 作品タイトル・説明
  • • メーカー・レーベル情報
  • • 再生時間・カテゴリー
  • • レビュー評価・マイリスト数
  • • 販売価格（通常版/HD版など）
  • • ランキング情報
  • • DUGA商品ページリンク
  • • アフィリエイト購入ボタン

🛒 DUGA購入方法

「今すぐ購入」ボタンから直接DUGAの商品ページに移動して購入できます。DUGAはポイント制での購入になります。

<\!-- プル・トゥ・リフレッシュ -->

⬇️ プル・トゥ・リフレッシュ

🔄 機能概要

画面を下方向に強く引っ張ることで、新しいDUGA動画を読み込むことができます。

使用場面

  • • 5動画制限時: ボタンを押さずに次の動画群を読み込み
  • • 新着動画更新時: 最新のDUGA動画を取得
  • • 接続エラー時: 再読み込みを実行
<\!-- DUGA独自機能 -->

🟣 DUGA独自機能

🎯 2023年以降優先表示

DUGA Swipeでは2023年以降の新しい作品を70%の割合で優先的に表示します。

🔄 完全ランダム再生

同じ動画の重複を避け、毎回異なる動画セットをランダムに表示します。

⚡ 高速キャッシュシステム

FirestoreとAPI組み合わせでDUGA APIの60req/60秒制限内で最適なパフォーマンスを実現。

🏷️ 豊富なカテゴリー

M男、アイドル、フェチ、個人撮影など、DUGAならではの多様なカテゴリーから検索可能。

<\!-- 便利な使い方のコツ -->

💡 便利な使い方のコツ

⚡ 効率的な視聴方法

  • • 個人撮影・アマチュア作品が豊富なのでじっくり探す
  • • 気に入った動画は作品詳細で完全版を確認
  • • カテゴリー検索で好みのジャンルを絞り込み
  • • プル・トゥ・リフレッシュで新着動画をチェック

🔧 DUGA特有の対処法

  • • 動画が読み込まれない → API制限の可能性、時間をおく
  • • 同じ動画ばかり → プル・トゥ・リフレッシュでランダム化
  • • 古い動画が多い → 検索で年代を絞り込み
  • • 購入方法が不明 → DUGAはポイント購入制
